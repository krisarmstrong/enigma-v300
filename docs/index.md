## Overview

Enigma v300 is a comprehensive implementation of the Enigma cipher machine in multiple languages:

- **Python (Functions)** - Functional programming approach
- **Python (Classes)** - Object-oriented approach
- **C** - C11 implementation
- **C++** - C++20 implementation

## Features

- Authentic Enigma machine simulation
- Multiple rotor configurations
- Plugboard support
- Historical accuracy
- Cross-platform compatibility
